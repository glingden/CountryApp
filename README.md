# Country Web App 
This web app provides you World Countries' information such as population, lanugages, borders, etc. In addition, 4 different colors of choice are available in the app.  This app is developed using React.js, React hooks, Redux and Material UI. <br><br> App is live at
https://graceful-biscotti-0acdbe.netlify.app/


## Running App 

1. Download the project either by clicking directly download option or git using following command:
```
$ git clone https://github.com/glingden/CountryApp.git
```

2. Navigate project folder and install required depedencies using following command:
  ```
  $ cd CountryApp
  $ npm install
  ```
3. Run App using following command;
  ```
  $ npm start
  ```
